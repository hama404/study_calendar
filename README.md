# STUDY CALENDAR

自分の勉強時間を可視化するアプリです。  
勉強の予定を立てることができます。

Ruby on Rails & REACT の Web アプリケーションです。  
フロントを REACT で作成して、SPA 化しました。  
Rails の Devise を使用してログイン機能を実装しています。  
(Ruby on Rails & REACT, CRUD, SPA)
